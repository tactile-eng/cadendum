Cadence NVDA Addon

Instructions

After installing addon, connect your device(s) through USB or Bluetooth and restart NVDA

Note: for all right-side devices, please pair the right-side device to the left-side device and then connect the left-side device to the computer through USB or Bluetooth

Make sure Cadence support is enabled by going to NVDA Prefences -> Settings -> Braille and ensuring that the braille display is set to "Automatic (Cadence HID Braille Display)"

If not, press Change, ensure "Cadence HID Braille Display" is checked, and select it from the dropdown

Usage

Device Orientation

For Quartet layout, press Row1 on each device to set the correct orientation for each device. Alternatively, press NVDA+Y to cycle through orientations.

For the Duet layout, switch between wide and tall by moving the devices to the correct orientation and pressing Row1 on each device. Alternatively, press NVDA+Y to cycle through orientations.

Image Mode

To toggle between image mode and text mode, press the Row3 button or NVDA+I

Image Mode buttons

While in image mode, you can use the following controls:

Direction Keys - pan image

Pan Left/Right - zoom out/in

Dot7 / Dot3 - increase / decrease threshold

Dot2 - reverse threshold

Dot1 - cycle color mode (grayscale / red / green / blue)

Row1 / Row2 + Direction keys - increase / decrease pan speed

Row1 / Row2 + Pan keys - increase / decrease zoom speed

Row1 / Row2 + Dot2 - increase / decrease threshold speed

Space + Direction keys - pan to edge of image

Dots1237 - Reset view
