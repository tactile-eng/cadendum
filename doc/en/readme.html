Cadence NVDA Addon

Instructions

After installing addon, connect your device(s) through USB or Bluetooth and restart NVDA

Make sure Cadence support is enabled by going to NVDA Prefences -> Settings -> Braille and ensuring that the braille display is set to "Automatic (Cadence HID Braille Display)"

If not, press Change, ensure "Cadence HID Braille Display" is checked, and select it from the dropdown

Usage

Image Mode

To toggle between image mode and text mode, press Space+dot2+dot4 or NVDA+I

Image Mode buttons

While in image mode, you can use the following controls:

Direction Keys - pan image

Pan Left/Right - zoom out/in

Row3 - reverse threshold

Ctrl + Up / Down - increase / decrease threshold

Row4 - cycle color mode (grayscale / red / green / blue)

Row1 / Row2 + Direction keys - increase / decrease pan speed

Row1 / Row2 + Pan keys - increase / decrease zoom speed

Row1 / Row2 + Row3 - increase / decrease threshold speed

Space + Direction keys - pan to edge of image

Center - lock focus

Row3 + Row4 - Reset view

Table Mode

To toggle between table mode and text mode, press space+dot1+dot2+dot4+dot5 or NVDA+g

Table mode buttons

While in table mode, you can use the following controls:

Direction Keys - move focus

Space+up/down - move by page

Center - exit

Rows - go to row and exit

PanLeft/PanRight - pan long text

Pan+up - toggle fixed column headers

Pan+left - toggle fixed row headers

Pan+center - toggle cell position headers
